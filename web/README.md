# Control — perform your life · 交互版

参考 @Livo《我如何创作》VOL.1 的制作技术栈(**Processing / p5.js 程序化动画**),
复刻并扩展「示例.mp4」的提线木偶 × 五线谱 × 飞鸟风格,后期录屏成片。

## 运行

```bash
tools/serve.py 8123
# 打开 http://localhost:8123/scenes/control/
# (根路径 / 是作品集画廊,本页是 control 的交互版)
```

## 玩法 / 交互

- **点击画面**开始:手升起、提线自指尖逐根甩出、五线谱绘入、标题敲入,
  随后约 60 个音符按顺序逐个弹出(同步拨弦声,音高按自然小调音级奏出)
- **移动鼠标扰动乐谱**:鼠标附近的谱线被拉扯抖动,受惊的音符化作飞鸟,
  **提线末端牵着鸟儿乱飞**(鸟儿扑翼纷飞);鼠标静止或移开后,鸟儿三三两两飞回原位变回音符
  (越到后面归队越慢,最后两只格外缓慢),谱面恢复平静
- **划谱如拨弦**:鼠标横向划过谱线时逐根发出琴弦声(五声音阶,音高随位置)
- **常驻飞鸟**:中间右侧有一只鸟永远保持鸟的形态,扑翅浮动、提线牵着,不随扰动变化
- 手整体 85% 半透明,提线末端拴在四根手指下方;署名 @GeekCatX
- 右上角控制条(悬停显示):`↻ 重播` / `音效 开关` / `旁白 开关`
- URL 参数:
  - `?auto=1` — **自动演示**:鼠标按预设轨迹扰动(5.5s–17.6s),随后自然归位,适合无人值守录制
  - `?t=5` — 静态渲染第 5 秒(配合 `&auto=1` 可截取惊飞状态,调试用)
  - `?mute=1` — 静音

## 曲谱规范

- 曲谱为伪随机生成(固定种子,画面可复现):5 行谱,每行 11-13 音,共约 60 音
- 音高位置按高音谱表(中线 = B4),随机游走、不出谱面;频率按 A 自然小调音级映射
- 符干方向:中线及以上朝下(符杠组统一由离中线最远的音决定)
- 符杠:2-4 音一组随机连杠,组间留拍隙;每行末音独立收束
- 每个音符各有一条红色提线连到指尖

## 飞鸟动画

- `assets/birds.json` 中 5 种燕子轮廓各是一种 wing pose,当作动画帧轮换
- 飞行中的鸟:约 8fps 快速扇翅,相位各自错开;常驻鸟:约 5fps 舒缓扇翅 + 浮动

## 配音 / 音效(纯 Web Audio 合成)

- **入场旋律**:音符弹出依次奏出真实音高(A4=440Hz 十二平均律)
- **暖 pad**:A2/E3/A3 持续垫底,音量随扰动强度起伏
- **惊飞**:下滑音拨弦 + 风声 whoosh;**归位**:轻柔拨弦
- **划谱拨弦**:鼠标穿越谱线触发五声音阶拨弦,像拨动琴弦
- **噪声微光**:随鼠标速度变化的"空气感"
- **旁白(可选)**:Web Speech 中文语音合成,默认关闭

## 录制

### 方式一:一条命令(hyperframes 离线渲染,可复现)

```bash
tools/make_film.sh record.html demo/control.mp4
```

脚本自动完成四步(细节见 ENGINE.md):

1. **渲染无声视频**:`record.html` 是 hyperframes 合成页,与主页面同一套场景,
   走确定性 seek 驱动(引擎 `window.__hfSeek` + paused GSAP 时间线)
2. **导出音轨事件**:headless Chrome 打开 `record.html?dump=1`,快进全片抓取 `__AUDIO__`
3. **离线合成配乐**:`tools/synth.py`(与 `engine/recipes.js` 同一套配方)
4. **混音成片**:ffmpeg 合成 `demo/control.mp4`

渲染模式下音轨事件(入场拨弦/惊飞/归位/划谱)与现场播放同一套触发逻辑,
同一固定种子 → 音画严格同步、每次渲染完全一致。

### 方式二:OBS 录屏(手动演奏)

- 画布内部分辨率 720×960(pixelDensity 2,Retina 下 1440×1920),与源片 3:4 竖屏一致
- `?auto=1` 自动演示 + OBS Studio「窗口采集」(可内录系统音频)
- 也可以手动模式亲自"演奏":鼠标扫过谱面惊起鸟群,静止后一切归位

## 素材与实现要点

- 代码结构:`engine/`(渲染契约/输入/音轨总线,通用)+ `scenes/control/scene.js`(本场景逻辑);
  引擎接口见根目录 ENGINE.md,新场景从 `templates/scene/` 复制起步
- `assets/hand.png` — 从原片帧提取的提线手剪影
- `assets/birds.json` — 从原片帧提取并矢量化的 5 种燕子(轮廓 + RDP 简化),缩放不模糊
- `lib/p5.min.js` — p5.js 1.11.13(与心法视频同版本),本地化离线可用
- 场景代码包在 IIFE 里,自定义函数避开 p5 保留名(`smooth`/`lerp`/`scale` 等)
